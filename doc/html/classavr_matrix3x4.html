Detailed Description

represents a 3x4 dimension matrix

This class simplifies the operations between 3x4 dimension matrices and offers auxiliary functions useful in renderization routines which use transformation matrices Internally, the matrix has 4x4 dimension, where the fourth dimension row, corresponds to a vector identity.
