Detailed Description

manages marker patterns with relations between them (calculated in real time)

Parameters
holderModedefinition mode of the base marker (responsible by renderization) [default is MODE_NEAR_PROJECTION] Possible modes are:
  • MODE_NEAR_PROJECTION base marker is the marker most near of the object projected
  • MODE_NEAR_CAMERA base marker is the marker most near of the real camera
  • MODE_RESISTENCE base marker is the marker which has less state change
  • MODE_INCLINATION base marker is the marker that is less inclined in relation to real camera
  • MODE_ACCURACY base marker is the marker that has the larger accuracy
  • MODE_PRIORITY base marker is the first visible marker (in the order of insertion, in other words, in the order shown in the config file)
