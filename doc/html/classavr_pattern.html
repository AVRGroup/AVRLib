Detailed Description

This class stores all information of a marker pattern which was registered in the application.

Stored Informations

  • identifier
  • visibility state
  • name (optional)
  • real width of the marker (in millimeters)
  • accuracy
  • center coordinates
  • final position matrix (internal use)
  • transformation matrix
