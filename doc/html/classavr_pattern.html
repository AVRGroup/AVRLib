Detailed Description

This class stores all information of a marker pattern which was registered in the application.

Stored Informations

  • identifier
  • visibility state
  • name (opcional)
  • real width of the marker (in millimeters)
  • accuracy
  • center coordinates
  • final position matrix (internal use)
  • transformation matrix
Parameters
filenamepath of the file that defines the marker pattern
patt_widthreal width of the marker (in millimeters)
patt_centercenter coordinates of the marker
